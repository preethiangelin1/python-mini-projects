# 📊 Crypto Price Tracker With SQLITE DB

A Python-based project that fetches live cryptocurrency prices, stores them in a SQLITE DB, and visualizes price trends using graphs.

---

## 🚀 Features

- 🔄 Fetches real-time crypto data using CoinGecko API  
- 💾 Stores data with timestamps in a SQLITE DB  
- 🧾 Displays top cryptocurrencies in the console  

---

## 🛠️ Technologies Used

- Python  
- requests  
- sqlite3

---

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/preethiangelin1/python-mini-projects.git
cd python-mini-projects/23_crypto_price_tracker_sqlite_db
```

2. Install dependencies:
```bash
pip install requests sqlite3
```

## Usage
```bash
python main.py
```
